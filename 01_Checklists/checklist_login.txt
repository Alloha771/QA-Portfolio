Чек-лист: Авторизация (Web)

Цель: убедиться, что форма логина работает корректно.

Проверки:
[ ] Страница логина загружается без ошибок
[ ] Поле Email принимает только валидный формат
[ ] Поле Пароль скрывает вводимые символы
[ ] Пустые поля → кнопка "Войти" неактивна
[ ] Неверный пароль → корректное сообщение об ошибке
[ ] Успешный вход с валидными данными
[ ] Ссылка "Забыли пароль?" ведёт на страницу восстановления
[ ] Флажок "Запомнить меня" сохраняет сессию
[ ] Выход очищает сессию
